Arrays / ArrayLists

Arrays

Arrays are a collection of values stored in a single variable. They allow information to be stored together rather than in individual variables.

To declare an array, place square brackets [] after the data type.

int[] numbers;

Initialization

An array can be initialized in two ways. Using an array literal, we can initialize the array by placing the comma-separated values between curly brackets. 

int[] numbers = {1,2,3,4,5};

Or we can initialize the array with a size and fill it later.

int[] numbers = new int[5];

Accessing Elements

You can access different elements of the array by referring to its index number in square brackets.

System.out.println(numbers[0]); //prints 1

It’s important to note that the first index is 0.

To change a value in an array, access the element and then use the assignment operator. 

numbers[2] = 1; //The array is now 1,2,1,4,5

Array Length

To get the number of elements in an array, access the array’s length property. 

System.out.println(numbers.length); //outputs 5

Using the length property we can also loop through every element in an array.

for (int i = 0; i < numbers.length; i++){
    System.out.println(numbers[i]);
}

For-Each Loop

A for-each loop or enhanced for loop is another way of looping through all the elements in an array. The enhanced for loop is only used to access the elements and you cannot directly modify the array elements. The format looks as follows:

for (type var: array){
    ...
}

The element for each iteration of the loop will be stored in the var. Here’s how it would look if we used it on our array from before.

for (int num: numbers){
    System.out.println(num);
}

Multidimensional Arrays

Arrays of arrays can also be made. We call these multidimensional arrays.

To create a 2-Dimensional array, we add an extra set of square brackets and when we initialize it as a literal, we add an extra set of curly braces.

int[][] numbers = { {1, 2, 3, 4}, {5, 6, 7, 8} };

Alternatively:

int[][] numbers = new int[2][4];

To access the elements, we’ll use two indexes, one to specify the array and the second to access the element in the array.

System.out.println(numbers[0][2]); //outputs 3

We can also use a nested for-loop to access every element in the 2-D array.

for (int i = 0; i < numbers.length; i++){
    for (int j = 0; j < numbers[i].length; j++){
        System.out.println(numbers[i][j]);
    }
}

ArrayList

ArrayLists have a similar function to arrays, but they have dynamic sizing, meaning they don’t have a preset size and will expand as more elements are added.

The ArrayList is found in the java.util package and can be imported by

import java.util.ArrayList;

All elements in ArrayLists are objects, meaning that an int ArrayList cannot be declared. For primitive data types, the equivalent wrapper class will have to be used. For ints, the wrapper class is Integer.

ArrayList<Integer> numbers = new ArrayList();

Other Methods

Listed below are the methods associated with ArrayLists that replace and add onto the functionality of the array.

  • .add(object) - used to add an object to the end of the ArrayList
  • .add(index, object) - used to add an object to a certain index of the ArrayList
  • .remove(index) - removes the object at the specified index, all the following elements are shifted
  • .get(index) - returns the object at the specified index
  • .set(index, object) - sets the element at the specified index to the object
  • .clear() - removes all elements in the ArrayList
  • .size() - returns the number of elements in the ArrayList
